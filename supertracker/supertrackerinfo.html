● Allows users to mark sighting coordinates on a map using a map API

● Used API documentation to build map functions with JavaScript, linking it to the HTML with Thymeleaf and Spring Java Controller

● Allows users to create, view, edit, and delete Sightings, Characters, Locations (using reverse GeoCoding), Organizations, and Powers

● Technologies Used: Java, Spring Boot, JDBC Template, MySQL, Thymeleaf, Leaflet, JavaScript
