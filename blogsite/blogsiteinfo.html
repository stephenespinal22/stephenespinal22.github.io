● Created a multi-page dynamic Spring Boot Full Stack Web Application using the MVC design pattern and incorporating a MySQL database

● Allows users to create, view, edit, and delete blog posts

● Blog Posts are able to be organized by category and hashtag

● Allows users to have different roles with security that hides website functionality based on role

● Technologies Used: Java, Spring Boot, JDBC Template, MySQL, Thymeleaf, JavaScript, Spring Security
